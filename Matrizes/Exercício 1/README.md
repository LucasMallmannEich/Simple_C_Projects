Utilizando Matrizes 

Proposta:

Elabore um programa que leia valores inteiros para uma matriz a[6][6].        
Calcule, a seguir, a soma dos elementos das colunas e linhas, escrevendo esses resultados num vetor.          
Escreva o vetor resultante.
